# KSx3267framework

KSX-3267 스마트온실 센서/구동기 노드 및 온실 통합제어기 간 인터페이스를 위한 프레임 워크입니다.

RS485 Modbus RTU 통신프로토콜 지원
SDK 지원
메타데이터및 Json 직렬화 지원
윈도우용 예제 프로그램 포함.


## Folder Structure


- `C_sharp`: .net Framework C# 용 소스
- `Docs`: SDK  도움말
- `Embedded`: 임베디드 시스템용 C  소스
- `Java`: JAVA 소스
- `Json`: 디폴트노드에 대한 메타데이터 json 파일
